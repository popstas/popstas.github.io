https://github.com/junegunn/fzf/wiki/examples

Удаляет сайты из nginx и apache:

find /etc/apache2/sites-enabled /etc/nginx/sites-enabled -type l | fzf --multi | xargs rm

Просмотр конфигов nginx:

find /etc/nginx/sites-enabled -type l | sort | fzf --preview="cat {}"

По идее это можно сделать через mc. Согласен, но когда у тебя в руках молоток вот команда, которую не заменит mc:

Удаляет выбранные остановленные контейнеры:

docker ps -a | fzf --multi | awk '{print $1}' | xargs docker rm
